# 🧠 Flashcard Quiz App

An interactive and user-friendly Flashcard Quiz Application designed to make learning more engaging and effective. Users can create custom flashcards, review study materials, and test their knowledge through an interactive quiz mode.

## 🚀 Features

✨ Create and manage flashcards easily
📚 Organize study content in a simple interface
🎯 Interactive quiz mode for self-assessment
🔄 Instant feedback on answers
💾 Data persistence using Local Storage
📱 Responsive design for desktop and mobile devices
🎨 Clean and modern user interface

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
